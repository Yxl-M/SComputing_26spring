Pair（密钥对）============================================================重要说明：- VM 通常默认只允许 “SSH 公钥登录”，所以必须准备 Key Pair。- Key Pair 包含：私钥（保存在本地） + 公钥（注入到 VM 的 authorized_keys）。2.1 创建 Key Pair（控制台操作）- 进入：计算（Compute）→ 密钥对（Key Pairs）- 创建一个 Key Pair，名称：SC4PS- 平台生成后显示/提供私钥2.2 将私钥文本保存为本地文件（macOS 实操）在本地终端执行：  mkdir -p ~/.ssh  nano ~/.ssh/SC4PS.pem在 nano 编辑器中，把平台显示的“私钥文本”完整粘贴进去（包含 BEGIN/END 行）：  -----BEGIN OPENSSH PRIVATE KEY-----  <此处为私钥内容，本文档不展示>  -----END OPENSSH PRIVATE KEY-----保存并退出 nano：- Ctrl + O  保存（Write Out）- 回车     确认文件名- Ctrl + X  退出设置私钥文件权限（非常重要，否则 ssh 会拒绝使用该私钥）：  chmod 600 ~/.ssh/SC4PS.pem备注（本次真实情况）：- 第一次 chmod 报错 “No such file or directory”，原因是当时文件还没正确保存成功；  重新执行 nano 并保存后，再 chmod 就正常了。============================================================3. VM。4.1 第一次登录 gate（强制修改 gate 密码）在本地终端执行：  ssh xyuan@gate.cloudveneto.it终端会提示输入密码：  (xyuan@gate.cloudveneto.it) Password: <输入初始 gate 密码，不回显>首次登录会强制修改密码，按提示依次输入：  Current password: <初始 gate 密码>  New password: <新密码>  Retype new password: <再次输入新密码>修改成功后exit退出：4.2 从本地通过 gate 直连 VM（推荐）命令格式：  ssh -J <gate_user>@gate.cloudveneto.it -i <private_key_path> <vm_user>@<vm_private_ip>本次实际成功命令（VM 用户名是 almalinux）：  ssh -J xyuan@gate.cloudveneto.it -i ~/.ssh/SC4PS.pem almalinux@10.67.20.70首次连接 gate 或首次连接 VM 时会出现 “host authenticity” 提示（正常）：  Are you sure you want to continue connecting (yes/no/[fingerprint])?输入：  yes之后会提示输入 gate 密码（因为通过 -J 走 gate）：  (xyuan@gate.cloudveneto.it) Password: <输入你刚设置的新 gate 密码>============================================================5. 在 AlmaLinux VM 上安装 C 编译环境并测试============================================================目标：安装 gcc/make 等工具链，编译运行一个 C 程序。5.1 更新系统 & 安装开发工具（在 VM 内执行）推荐安装 “Development Tools” 组（包含 gcc、make 等）：  sudo dnf update -y  sudo dnf groupinstall -y "Development Tools"验证 gcc 已安装：  gcc --version5.2 在 VM创建并编译运行代码：cat > hello.c <<'EOF'#include <stdio.h>int main() {  printf("Hello CloudVeneto!\n");  return 0;}EOFgcc hello.c -o hello./hello输出：Hello CloudVeneto!6.
